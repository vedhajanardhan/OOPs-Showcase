# OOPs-Showcase

This repository contains Java mini projects built to demonstrate Object-Oriented Programming concepts.  
Currently implemented: **Library Management System**

## Concepts Demonstrated
- Encapsulation: Book details are private with getters/setters.
- Inheritance: Student and Librarian extend User.
- Polymorphism: Overloaded search methods in Library.
- Abstraction: Abstract User class.

## How to Run
1. Clone the repository
2. Compile all .java files
   javac *.java
3. Run the main program
   java Main
